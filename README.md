# BatchUnZip

简单快捷的批量压缩包解压程序

### 设计功能

这是一个设计简单的小程序，或者说是“脚本”，主要目的是快速方便地大批量处理包含很多压缩包的文件夹。

比如，下载和整理文件资源、大量处理带有目录结构的归档、解压还原各种网盘下载的资源等等。

该程序可以自动扫描文件夹下的常见压缩包，并调用7-zip命令行进行快速解压，**然后删除原始压缩包。**

### 使用方法

```
BatchUnZip <Directory>
```
> 注：仅支持第一个参数中的目录名。

### 支持特性

- 自动处理zip、001、7z、rar压缩包并正确删除zip和001对应的zXX和0XX分卷。

- 自动检测密码，需要密码时用户需手动输入。密码错误不影响解压其他压缩包，且不会误删此带密码压缩包。

- 根据7-zip命令行exitcode判断解压情况，遇到文件损坏、格式未知压缩包可避免误删。

- （代码）支持快速地继续新增压缩包后缀格式，如果需要解压其他类型压缩包如gz、xz等，直接加入`ArchiveScanner.SupportedExtensions`即可

- 支持循环解压压缩包，直至嵌套压缩包都被解压彻底。（支持自动跳过解压失败的压缩包，避免误删/死循环。）

### 当前缺陷

- 解压完后复原文件夹路径时，如果被杀毒软件/其他扫描程序异常占用，文件夹移动可能会失败。

- 目前不支持自动处理覆盖，如果压缩包中存在同名文件会造成假死。
